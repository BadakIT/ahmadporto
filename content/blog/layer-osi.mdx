# Pengenalan Layer OSI

Model OSI (Open Systems Interconnection) adalah model konseptual yang
digunakan untuk memahami bagaimana data dikomunikasikan dalam jaringan komputer.
Model ini dikembangkan oleh International Organization for Standardization (ISO)
dan terdiri dari tujuh lapisan (layer). Setiap lapisan memiliki fungsi khusus dan bekerja
secara berurutan untuk memastikan komunikasi data yang efektif dan efisien.

Dengan memahami model OSI, kita dapat lebih mudah mengidentifikasi dan
menyelesaikan masalah dalam jaringan, serta memahami bagaimana perangkat jaringan
berinteraksi satu sama lain.

# 7 Layer Dalam Model OSI

<img src="/layerosi.jpg" alt="Layer OSI" width={500}/>

1. Application Layer (Layer ke 7)

Layer ini menjadi pusat interaksi antara user denga aplikasi yang digunakan menggunakan fungsionalitas
jaringan. Layer ini menjadi layer paling atas, beberapa contoh Application layer adalah HTTP,FTP,SMPT, dan lain lain.

2. Presentation layer (Layer ke 6)

Berfungsi untuk mengidentifikasi sintaks yang dipakai suatu host jaringan. Layer ini perlu memberi enkripsi
serta deskripsi data yang nantinya akan di pakai dalam layer application. Pada layer ini data ter-enkripsi
otomatis melalui sistem. ada beberap protokol di layer ini seperti SSL.

3. Session Layer (Layer ke 5)

Lapisan session berperan dalam mengelola dialog dan koneksi antara komputer. Selain itu, lapisan
ini juga dapat mengatur pemutusan koneksi internet pada suatu perangkat. Beberapa protokol yang bekerja
pada lapisan ini antara lain NFS, RTP, dan SMB.
